概要

概要

授業前に授業の情報を教えてくれるLINE bot。

一般用に開発した くらりん（クラスリマインダー）の前身。

Qiita にて 「文系でも分かる、GASによる授業通知のLINE bot の作成」掲載。

作成動機

大学生である私は、オンライン授業を受ける中で、毎回zoomのIDやパスワードを探すのが大変だと思った。そこで、LINE botで10分前に授業の情報の通知が来たら便利だという考えた。

技術仕様

GAS, Messaging API
